# BestCommerceProductDiscounts
 
How to run applicatoin:
1. Build docker image. 
2. Run docker-compose. 
3. Create .zip from `Discount`
4. Open .zip in IIB. 
5. Deploy on localhost. 
6. Use `Discount.postman_collection.json` to get data. 

Due to limited time, I haven't protect against SQLInjection. I realise that, so next step It will be to improve this.
